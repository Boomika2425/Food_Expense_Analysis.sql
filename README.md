# 🍽️ Food Expense Analysis using MySQL

## 📌 Project Overview

This project analyzes personal food expenses using MySQL. It demonstrates SQL concepts such as table creation, data insertion, aggregation, grouping, sorting, and reporting to generate meaningful insights from spending data.

---

## 🛠️ Tools Used

- MySQL
- SQL

---

## 📂 Database Structure

Database:
- food_expense_analysis

Table:
- food_expenses

Columns:
- expense_id
- expense_date
- category
- amount

---

## 📊 SQL Concepts Used

- CREATE DATABASE
- CREATE TABLE
- INSERT
- SELECT
- GROUP BY
- ORDER BY
- SUM()
- AVG()
- LIMIT
- DAYNAME()

---

## 📈 Analysis Performed

### 1. Weekly Food Spending
Calculated total spending for each day of the week.

### 2. Highest Expense Transaction
Found the highest single food expense.

### 3. Lowest Expense Transaction
Found the lowest single food expense.

### 4. Highest Spending Category
Calculated total spending by category and identified the highest.

### 5. Lowest Spending Category
Calculated total spending by category and identified the lowest.

### 6. Average Daily Expense
Calculated average spending by day and category.
